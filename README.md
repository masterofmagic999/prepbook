# prepbook

A curated study guide and preparation resource for technical interviews and coding assessments.

## Overview

**prepbook** is an open collection of notes, problem breakdowns, and study material designed to help developers prepare for technical interviews. It covers common data structures, algorithms, system design concepts, and behavioural interview tips.

## Contents

- **Data Structures** – Arrays, Linked Lists, Stacks, Queues, Trees, Graphs, Hash Maps, and more
- **Algorithms** – Sorting, Searching, Dynamic Programming, Recursion, Greedy, Backtracking
- **System Design** – Scalability fundamentals, databases, caching, load balancing, and real-world design examples
- **Behavioural** – STAR-method answers, common questions, and tips for culture-fit interviews
- **Language Cheatsheets** – Quick reference snippets for Python, JavaScript, Java, and more

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/masterofmagic999/prepbook.git
   cd prepbook
   ```
2. Browse the topic folders or use your editor's search to find a specific subject.
3. Work through problems, take notes, and add your own solutions.

## Contributing

Contributions are welcome! If you spot an error, want to add a new topic, or improve an existing explanation:

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/your-topic`)
3. Commit your changes (`git commit -m 'Add notes on topic X'`)
4. Push to your branch and open a Pull Request

Please keep entries concise, accurate, and beginner-friendly where possible.
